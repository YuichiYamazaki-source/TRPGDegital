# Call of Cthulhu 7th Edition — Character Creation Rules

> Source: Call of Cthulhu Quick-Start Rules, 7th Edition (Chaosium Inc.)
> This document is for the character creation support agent (pre-play phase).

---

## 1. Characteristics (8 stats)

| Stat | Full Name    | Description                            |
|------|-------------|----------------------------------------|
| STR  | Strength     | Physical power                         |
| CON  | Constitution | Health and resilience                  |
| SIZ  | Size         | Height and weight combined             |
| DEX  | Dexterity    | Agility and speed                      |
| APP  | Appearance   | Attractiveness and charisma            |
| INT  | Intelligence | Cleverness, logic, and intuition       |
| POW  | Power        | Willpower, mind, and mental stability  |
| EDU  | Education    | Formal schooling or life experience    |

**Stat assignment (Quick-Start):** Distribute these values among the 8 stats: 40, 50, 50, 50, 60, 60, 70, 80.

Each stat has three values:
- **Full value** (e.g. 60)
- **Half value** (Hard): Full / 2, round down (e.g. 30)
- **Fifth value** (Extreme): Full / 5, round down (e.g. 12)

## 2. Derived Attributes

| Attribute         | Calculation                                    |
|-------------------|------------------------------------------------|
| Damage Bonus (DB) | Based on STR + SIZ (see table below)           |
| Build             | Based on STR + SIZ (see table below)           |
| Hit Points (HP)   | (CON + SIZ) / 10, round down                  |
| Movement Rate     | 8 (for all human characters)                   |
| Sanity (SAN)      | Equal to POW at game start                     |
| Magic Points (MP) | Equal to POW / 5                               |
| Luck              | Determined separately (2D6+6) * 5 or assigned  |

**Damage Bonus & Build Table:**

| STR + SIZ | Damage Bonus | Build |
|-----------|-------------|-------|
| 2-64      | -2          | -2    |
| 65-84     | -1          | -1    |
| 85-124    | +0          | 0     |
| 125-164   | +1D4        | 1     |
| 165-204   | +1D6        | 2     |

## 3. Occupations & Skill Assignment

### 3.1 Occupation Skill Points

Assign to 8 occupation skills + Credit Rating:
- 1 skill at 70%
- 2 skills at 60%
- 3 skills at 50%
- 3 skills at 40%

### 3.2 Personal Interest Skills

Choose 4 non-occupation skills and add 20% to each base value.

### 3.3 Sample Occupations

| Occupation           | Skills                                                                                     |
|----------------------|--------------------------------------------------------------------------------------------|
| Antiquarian          | Appraise, Art/Craft (any), Library Use, Spot Hidden, History, 1 Interpersonal, Other Language, 1 any |
| Author               | Occult or Natural World, Art/Craft (Literature), Psychology, Library Use, History, Other Language, Own Language, 1 any |
| Dilettante           | Art/Craft (any), Firearms, Ride, 1 Interpersonal, Other Language, 3 any                   |
| Doctor               | Medicine, First Aid, Science (Biology), Science (Pharmacy), Psychology, Other Language (e.g. Latin), 2 any |
| Journalist           | Art/Craft (Photography), Psychology, Library Use, History, 1 Interpersonal, Own Language, 2 any |
| Detective            | Listen, Art/Craft (Acting) or Disguise, Firearms, Psychology, Law, Spot Hidden, 1 Interpersonal, 1 any |
| Private Investigator | Art/Craft (Photography), Psychology, Library Use, Disguise, Law, Spot Hidden, 1 Interpersonal, 1 any |
| Professor            | Psychology, Library Use, Other Language, Own Language, 4 any (research/personal specialties) |

**Custom occupations:** Choose any profession, pick 8 skills appropriate for it, and get Keeper approval.

**Interpersonal skills:** Intimidate, Fast Talk, Persuade, or Charm (pick one where noted).

### Skill Descriptions

**Combat:** Fighting (Brawl) covers unarmed combat plus knives/clubs/simple weapons. Dodge lets you avoid attacks (can dodge multiple times per round, once per attack).

**Investigation:** Spot Hidden finds hidden objects and clues. Listen interprets sounds. Library Use finds specific info in collections (takes hours). Psychology reads motivations and detects lies.

**Interpersonal:** Intimidate frightens/coerces. Fast Talk deceives verbally. Persuade uses logical argument (30+ min). Charm uses attractiveness/flattery. All are opposed by themselves or Psychology.

**Knowledge:** Occult recognizes occult lore. History recalls events. Science covers specialties (Biology, Chemistry, etc.). Accounting tracks finances. Archaeology dates artifacts.

**Practical:** First Aid (within 1 hour, restores 1 HP). Medicine (1+ hour, restores 1D3 HP). Locksmith picks locks. Stealth moves silently. Climb/Jump/Swim/Drive/Navigate/Survival/Track are self-explanatory.

**Special:** Credit Rating measures wealth (0%=destitute, 10-49%=average, 50-89%=wealthy, 90-98%=rich, 99%=super-rich). Cthulhu Mythos cannot be assigned at creation. Psychoanalysis treats insanity. Own Language base = EDU.

## 5. Backstory

Fill in a few short entries on the investigator sheet (not all are required):
- **Appearance:** How others see the investigator.
- **Ideology/Beliefs:** What the investigator believes in or their outlook on life.
- **Traits:** Distinctive habits or behaviors.
- **Significant People:** Important relationships.
- **Treasured Possessions:** Prized items.
- **Equipment:** Standard gear appropriate for the occupation.

## 6. Dice Notation Reference

| Notation | Meaning                                    |
|----------|--------------------------------------------|
| D100     | Percentile dice (two D10s)                 |
| D10      | 10-sided die (1-10)                        |
| D8       | 8-sided die                                |
| D6       | 6-sided die                                |
| D4       | 4-sided die                                |
| D20      | 20-sided die                               |
| 2D6      | Roll 2 six-sided dice, sum the results     |
| 1D6+1    | Roll D6, add 1 to the result               |

- D100 reading: "tens" die shows 00 and "units" die shows 0 = **100**.
- "tens" die shows 00 and "units" die shows non-zero = single digit (e.g., 00+3 = 3).
